1번 문제는 비슷한 숫자 끼리의 뺄셈 입니다.
분산을 구하는 두 가지 식을 작성하여 컴퓨터상에서 두 식이 얼마나 다른지 실험하는데
실험을 하기 앞서 cpp 파일의 44번 줄에서 실험에 필요한 데이터를 생성합니다.
매개변수로 1을 넘기면 실험하기에 적절한 정해진 데이터를 생성하며
매개변수로 0을 넘기면 실험에 관계없이 난수를 생성합니다.
이후 생성한 실험 데이터로 실험을 진행합니다. 두 식의 시간을 측정하고 결과를 비교합니다.

2번 문제도 비슷한 숫자 끼리의 뺄셈 입니다.
2차 방정식( f(x) = ax^2 + bx + c = 0 )에서 근의 공식을 사용하여 두 근을 구할 때,
a와 c가 b보다 상대적으로 많이 작은 값이면 근의 공식( ( -b +- root(b^2 - 4ac) )/2a )을 이용해 두 근을 구할 때 비슷한 숫자 끼리의 뺄셈을 하는데
여기서 수치적인 오차( -b + root(b^2 - 4ac) )가 발생합니다. 따라서 이 문제를 해결하기 위해 근의 공식에서 유리화를 하여 근을 구합니다.
적절한 입력으로는
1) a : 1, 		b : 10000,	c : 1
2) a : 0.0001, 	b : 9999, 	c : 0.0001
등이 있습니다.
